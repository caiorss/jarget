  6.  def downloadArtifact(pack: PackData, ext: String, path: String): Reader[String, Unit]

    Download artifact with extension jar or pom to a given path.

    Download artifact with extension jar or pom to a given path.

    pack

    - Package that will be download

    ext

    - File extension - 'jar', 'pom', 'md5', 'sha256' and etc.

    path

    - Path where the artifact will be stored. Example: It will download the package's jar artifact and store it in ./download/jfreechart-1.0.17.jar

    val p    = PackData("org.jfree", "jfreechart", "1.0.17")
val repo =  "http://central.maven.org/maven2"
val path = "./downloads"
Packget.downloadArtifact(p, "jar", path) run(repo)
